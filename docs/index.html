Le dojo Programmation de Paris

Le Dojo Programmation de Paris est un lieu où des développeu-r-seuse-s se retrouvent régulièrement pour programmer ensemble sur un exercice, s'amuser et pratiquer de façon délibérée afin d'améliorer leurs compétences.

Nous historisons le code produit sur le dépôt.

Nous nous concentrons sur la programmation en groupe, le plus souvent en partant de zéro, pendant une courte durée (~1h30). Nous utilisons des langages, des outils et des formats d'exercices variés.

Pas besoin d'ordinateur, amenez votre bonne humeur !

Plus de détails sur ce que nous faisons dans le wiki du dojo.

Pour des sujets plus théoriques sur le développement, nous vous invitons à rejoindre le groupe Software Crafters Paris.

Créé en 2004, c'est le premier dojo de programmation de l'histoire.

Nous utilisons ce groupe pour nous retrouver.
