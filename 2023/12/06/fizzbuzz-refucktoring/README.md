# Some code smells

## General
- Duplication
- Dead code
- Comments
- Non indent code

## Function
- Long body
- Long parameter list
- Input/Output parameters

## Variables and fields
- Non const
- Temporary

## Conditional
- Negative
- Embedded
- Overlapping

## Naming :
- Abbreviation
- Non-sense
- No respect for calling convention
